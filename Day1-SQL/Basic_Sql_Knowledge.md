#Creating a Database 
```
CREATE DATABASE database_name;
```
#Using a Database 
```
USE database_name;
```
#List all the tables in the database
```
SHOW TABLES
```
#List tables on specific names
<!--  After like we use wildcard operators so please read them to use them. -->
```
SHOW TABLES LIKE '%string-to-be-matched%' 
```
